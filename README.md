# Today Project


# MongoDB Cluster
Database mongodb Atlas
# Requirements

- Java JDK 8 to 14.
- Maven 3.6.3.

# Command lines

- Compile: 

```sh
mvn clean compile
```

- Run the `Server` class: 

```sh
mvn compile exec:java -Dexec.mainClass="fr.alma.today.Server"
```

- Run the `Client` class: 

```sh
mvn compile exec:java -Dexec.mainClass="fr.alma.today.Client"
```
